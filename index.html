Privacy Policy

Application Information

App Name: Telecom Sentiment Analysis

Purpose: Academic research project for Master's degree in Data Visualization

Institution: University of Algiers 1 - M2 ASD

Data Collection

This application collects ONLY publicly available data from Facebook pages:

  • Public posts from official telecom pages (Ooredoo, Djezzy, Mobilis)
  • Public comments from these posts
  • Public engagement metrics (reactions, shares)

Data Usage

Collected data is used exclusively for:

  • Academic research and analysis
  • Sentiment analysis visualization
  • Customer satisfaction comparison study

Data Storage & Security

  • Data is stored locally in CSV files
  • No personal user data is collected
  • All data is anonymized
  • Data will be deleted after project completion (June 2024)
